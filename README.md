## What FAF Enterprise is

Persistent Project Context for teams, large orgs, monorepos — IANA-registered .faf format · 33 slots, ELv2 license

Part of the **FAF Family** — the IANA-registered Foundational Context Layer for AI. `.faf` is the format ([`application/vnd.faf+yaml`](https://www.iana.org/assignments/media-types/application/vnd.faf+yaml)), and FAF Enterprise is one of the products that builds on it.
