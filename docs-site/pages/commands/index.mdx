# Commands Reference

Complete reference for all FAF CLI commands.

## Core Commands

### [chat](./chat)
Natural language .faf generation - conversation-driven context building
```bash
faf chat
```

### [check](./check)
Comprehensive .faf validation and freshness check
```bash
faf check [--format] [--fresh] [--fix] [--detailed]
```

### [init](./init)
Create .faf file from your project (detects React, Python, Node.js, etc.)
```bash
faf init [directory] [--force] [--template react|python|node]
```

### [score](./score)
Rate your .faf completeness (0-100%). Aim for 🥉85% BRONZE+ for solid AI context you can build on
```bash
faf score [file] [--details] [--minimum 80]
```

### [status](./status)
Quick .faf context health status (<200ms)
```bash
faf status
```

### [sync](./sync)
Update .faf when dependencies change OR sync with claude.md
```bash
faf sync [file] [--bi-sync] [--auto]
```
