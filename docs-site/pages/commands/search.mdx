# search

Search within .faf file content with intelligent highlighting

## Usage

```bash
faf search <query> [--section project] [--case] [--keys] [--values] [--count]
```


## Examples

```bash
faf search "react"
```
```bash
faf search "api" --section project
```
```bash
faf search "test" --keys
```
