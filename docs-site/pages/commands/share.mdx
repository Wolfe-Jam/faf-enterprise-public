# share

Secure .faf distribution with auto-sanitization and multiple formats

## Usage

```bash
faf share [file] [--private] [--format yaml|json|url] [--expires 7d] [--anonymous]
```


## Examples

```bash
faf share
```
```bash
faf share --anonymous
```
```bash
faf share --format json --expires 24h
```
