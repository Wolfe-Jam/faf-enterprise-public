# todo

Claude-inspired todo system - gamify your context improvements

## Usage

```bash
faf todo [--show] [--complete <task>] [--reset]
```


## Examples

```bash
faf todo
```
```bash
faf todo --show
```
```bash
faf todo --complete 1
```
```bash
faf todo --complete readme
```





## Category

**improvement** - Context improvement tools
