# trust

Unified trust dashboard with confidence, garage, panic, quality modes

## Usage

```bash
faf trust [--detailed] [--confidence] [--garage] [--panic] [--quality]
```


## Examples

```bash
faf trust
```
```bash
faf trust --confidence
```
```bash
faf trust --garage
```
```bash
faf trust --panic
```





## Category

**trust** - Trust and confidence management
