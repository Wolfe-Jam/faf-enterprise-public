# verify

Test .faf context with Claude, ChatGPT & Gemini - prove AI understanding

## Usage

```bash
faf verify [--detailed] [--models claude,chatgpt,gemini] [--timeout 30000]
```


## Examples

```bash
faf verify
```
```bash
faf verify --detailed
```
```bash
faf verify --models claude,chatgpt
```
